Import Scanner untuk digunakan untuk membaca input dari pengguna,  Input n untuk pengguna memasukkan bilangan bulat positif,  Validasi untuk jika n &lt; 0 tampilkan pesan error,  Perulangan for untuk menghitung faktorial dengan mengalikan nilai faktorial secara bertahap, Output hasil untuk menampilkan nilai faktorial yang sudah dihitung.
